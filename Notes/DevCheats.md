# Dev Cheat Codes

Type these while on the fishing screen (no input field needed, just press the keys):

| Code     | Effect                                      |
|----------|---------------------------------------------|
| `lol`    | +1,000,000 gold                             |
| `gold`   | Spawn a golden fish                         |
| `frenzy` | Trigger Feeding Frenzy (rapid spawns, 10s)  |
| `boon`   | Trigger Fishmonger's Boon (3x sell, 10s)    |
| `chest`  | Trigger Treasure Chest (all golden, 10s)    |
| `upgrades`| Max all upgrades + 10 of each consumable   |
